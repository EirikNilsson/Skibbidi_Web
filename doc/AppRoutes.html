Overview

Modulen AppRoutes ger hjälpmetoder för att definiera HTTP-rutter (GET och POST) samt en metod för att skapa HTTP-omdirigeringar.

Den förutsätter att ett ‘router`-objekt är tillgängligt och kan ta emot rutter.

def redirect(location, status: 302)
  [
    status,
    { "Location" => location, "Content-Type" => "text/html" },
    "<html><body>Redirecting to <a href='#{location}'>#{location}</a></body></html>"
  ]
end
